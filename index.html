Política de Privacidade — PLEXOR

Última atualização: 23 de abril de 2026

Esta política descreve como o jogo PLEXOR, desenvolvido por Frydarka Studios (Guilherme Ribeiro de Castro), coleta e usa informações dos usuários.

Informações Coletadas

O PLEXOR coleta apenas o nome de jogador escolhido voluntariamente pelo usuário para exibição no ranking global. Nenhuma informação pessoal identificável é coletada sem consentimento explícito.

Uso das Informações

O nome de jogador é utilizado exclusivamente para exibição no ranking global de pontuações, armazenado em servidores seguros do Supabase (supabase.com).

Armazenamento de Dados

As pontuações e nomes de jogadores são armazenados em banco de dados seguro. O usuário pode solicitar a exclusão dos seus dados a qualquer momento.

Compartilhamento de Dados

Não compartilhamos dados pessoais com terceiros para fins comerciais ou publicitários.

Anúncios

O PLEXOR não exibe anúncios e não utiliza redes de publicidade.

Crianças

O PLEXOR não é direcionado a crianças menores de 13 anos e não coleta dados de menores intencionalmente.
